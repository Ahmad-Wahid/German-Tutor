# 🇩🇪 Lena - German Language Tutor (A2)

A friendly German conversation partner built as a single-page web app. Practice German speaking with an AI tutor that corrects your mistakes and keeps the conversation flowing.

## Features

- 🗣️ **Voice Input** - Speak German and get instant feedback (requires HTTPS)
- 🤖 **AI Tutor** - Uses Groq or Gemini for natural conversations
- 📝 **Grammar Corrections** - Gentle corrections for your German
- 🔊 **Text-to-Speech** - Listen to native-like German pronunciation
- 📱 **Mobile Friendly** - Works on your phone browser

## How to Use

1. Get a free API key from:
   - [Groq Console](https://console.groq.com) - 14,400 requests/day free
   - [Google AI Studio](https://aistudio.google.com) - 1,500 requests/day free
2. Enter your API key in the settings (gear icon)
3. Start speaking or typing in German!

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- Web Speech API (Speech Recognition & Synthesis)
- Groq or Gemini AI APIs

## Local Development

Just open the HTML file in a browser, or serve it with:

```bash
python3 -m http.server
```
